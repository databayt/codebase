## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `providers` | `OAuthProvider[]` | `["github", "google", "apple", "paypal"]` | OAuth providers to display |
| `columns` | `1 \| 2` | `2` | Number of columns in grid |
| `className` | `string` | - | Additional CSS classes |
