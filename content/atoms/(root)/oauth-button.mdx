## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `provider` | `"github" \| "google" \| "apple" \| "paypal"` | required | The OAuth provider |
| `label` | `string` | Provider name | Custom button label |
| `variant` | `ButtonVariant` | `"outline"` | Button variant |
| `className` | `string` | - | Additional CSS classes |

## Accessibility

- Uses semantic button element
- Includes provider icon with proper aria labels
- Keyboard navigable
- Focus visible states
