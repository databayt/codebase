## Usage

```tsx
import { PaymentMethodSelector } from "@/components/atom/payment-method-selector"
```

```tsx
<PaymentMethodSelector
  value="card"
  onValueChange={(method) => console.log(method)}
  methods={["card", "paypal", "apple"]}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `"card" \| "paypal" \| "apple"` | `"card"` | Selected method |
| `onValueChange` | `(method) => void` | - | Selection handler |
| `methods` | `PaymentMethod[]` | `["card", "paypal", "apple"]` | Available methods |
