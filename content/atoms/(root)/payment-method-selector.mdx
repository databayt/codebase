## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `"card" \| "paypal" \| "apple"` | `"card"` | Selected payment method |
| `onValueChange` | `(method: PaymentMethod) => void` | - | Selection change handler |
| `methods` | `PaymentMethod[]` | `["card", "paypal", "apple"]` | Available payment methods |
| `className` | `string` | - | Additional CSS classes |

## Accessibility

- Uses radio group pattern for single selection
- Keyboard navigable with arrow keys
- Clear visual selection state
- Screen reader announces selected payment method
