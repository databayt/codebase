## Run Localhost

**a. Install VS Code**

Download from [code.visualstudio.com](https://code.visualstudio.com/)

**b. Install Node.js**

Download from [nodejs.org](https://nodejs.org/) (LTS version)

**c. Install Git**

Download from [git-scm.com](https://git-scm.com/downloads)

**d. Open VS Code**

Launch VS Code, then open the terminal: `Ctrl + `` (backtick) or View → Terminal

**e. Install pnpm**
```bash
npm install -g pnpm
```

**f. Clone the repo**
```bash
git clone https://github.com/databayt/hogwarts.git
```

**g. Open project in VS Code**

File → Open Folder → select the `hogwarts` folder

**h. Open terminal in VS Code**

`Ctrl + `` (backtick) or View → Terminal

**i. Install dependencies**
```bash
pnpm install
```

**j. Create environment file**
```bash
cp .env.example .env
```

**k. Generate Prisma client**
```bash
pnpm prisma generate
```

**l. Setup database**
```bash
pnpm prisma migrate dev
```

**m. Start dev server**
```bash
pnpm dev
```

**n. Open browser**

Go to http://localhost:3000

_For detailed setup, see [Localhost Setup](/docs/localhost)._

## Submit a PR

```bash
git checkout -b feature/your-name
```

```bash
git add .
```

```bash
git commit -m "feat: add your change"
```

```bash
git push origin feature/your-name
```

_For full contribution guide, see [Contributing](/docs/community) (coming soon)._

## First Contribution

Add your name to our contributors showcase:

**1. Create a new file:** `src/app/[lang]/(marketing)/blog/contributors/YOUR-NAME.mdx`

**2. Add your info:**

```mdx
---
name: Your Name
github: your-github-username
date: 2024-11-21
---

# Your Name

Hi! I'm contributing to Hogwarts because [your reason].
```

**3. Submit PR** with title: `feat: add [Your Name] to contributors`

**That's it! Welcome to Hogwarts.** _Join our community of contributors building the future of school automation._
