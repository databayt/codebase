# Get Started

## 1. The Idea

Building a school automation platform like these real-world products:
- [PowerSchool](https://www.powerschool.com) - $2.8B valuation, 45M students
- [Skyward](https://www.skyward.com) - 5.7M students, 1,900+ districts
- [Infinite Campus](https://www.infinitecampus.com) - 7.8M students, 50 US states
- [Classter](https://www.classter.com) - 500K+ students, 40+ countries

## 2. Run Localhost

Quick setup in 5 minutes:

```bash
git clone https://github.com/databayt/hogwarts.git && cd hogwarts
```

```bash
pnpm install
```

```bash
cp .env.example .env
```

```bash
pnpm prisma generate && pnpm prisma migrate dev
```

```bash
pnpm dev
```

**Running at:** http://localhost:3000

## 3. Submit a PR

```bash
git checkout -b feature/your-name
```

```bash
git add . && git commit -m "feat: add your change"
```

```bash
git push origin feature/your-name
```

## 4. First Contribution

Add your name to our contributors showcase:

**1. Create a new file:** `src/app/[lang]/(marketing)/blog/contributors/YOUR-NAME.mdx`

**2. Add your info:**
```mdx
---
name: Your Name
github: your-github-username
date: 2024-11-21
---

# Your Name

Hi! I'm contributing to Hogwarts because [your reason].
```

**3. Submit PR** with title: `feat: add [Your Name] to contributors`
