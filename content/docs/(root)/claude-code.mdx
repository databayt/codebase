Get Claude Code running with the `c` alias for permission-free development.

## Installation

### macOS / Linux

```bash
curl -fsSL https://claude.ai/install.sh | sh
```

### Windows (PowerShell)

```powershell
irm https://claude.ai/install.ps1 | iex
```

### Verify

```bash
claude --version
```

## The `c` Alias

Add to `~/.zshrc` (or `~/.bashrc`):

```bash
alias c='claude --dangerously-skip-permissions'
```

Reload shell:

```bash
source ~/.zshrc
```

Now use `c` instead of `claude`:

```bash
c              # Start session
c "fix bug"    # Direct prompt
c -r           # Resume last session
```

## User-Level Directory

```
~/.claude/
├── CLAUDE.md           # Global instructions
├── settings.json       # Environment & hooks
├── commands/           # Custom slash commands
├── agents/             # Custom agents
└── bin/                # Binaries (auto-managed)
```

## CLAUDE.md (Global Instructions)

Create `~/.claude/CLAUDE.md`:

```markdown
# Global Claude Code Instructions

## Preferences
- **Package Manager**: pnpm
- **Stack**: Next.js, React, TypeScript, Tailwind

## Behavior
- Use TypeScript strict mode
- Prefer Server Components
- Follow atomic design patterns
```

## settings.json

Create `~/.claude/settings.json`:

```json
{
  "env": {
    "CODEBASE_PATH": "/path/to/your/codebase"
  },
  "hooks": {
    "PreToolUse": [
      {
        "matcher": "Bash(pnpm dev)",
        "hooks": [
          {
            "type": "command",
            "command": "lsof -ti:3000 | xargs kill -9 2>/dev/null; exit 0"
          }
        ]
      }
    ]
  }
}
```

## Shell Configuration

Complete `~/.zshrc` example:

```bash
# Claude Code
export PATH="$HOME/.claude/bin:$PATH"
alias c='claude --dangerously-skip-permissions'
```

## Permission Flags

| Flag | Purpose |
|------|---------|
| `--dangerously-skip-permissions` | Skip all prompts |
| `--allowedTools "Bash,Read,Write"` | Whitelist tools |
| `--disallowedTools "WebFetch"` | Blacklist tools |
| `-p "prompt"` | Direct prompt mode |
| `-r` | Resume last session |

## Global vs Project Config

| Location | Scope | Priority |
|----------|-------|----------|
| `~/.claude/CLAUDE.md` | All projects | Lower |
| `./CLAUDE.md` | Current project | Higher |
| `~/.claude/settings.json` | All projects | Lower |
| `./.claude/settings.json` | Current project | Higher |

Project config overrides global config.
