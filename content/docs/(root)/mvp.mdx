## Epic 01: Infrastructure

Build the scalable foundation for a multi-tenant SaaS platform with subdomain-based routing (school.databayt.org).
Implement Next.js 15 middleware for tenant detection, i18n routing (Arabic RTL/English LTR), and JWT session context injection.
Engineer Prisma 6 multi-tenant isolation where every query is scoped by schoolId with unique constraints preventing cross-tenant data leakage.
Master CI/CD pipelines, Sentry error monitoring, and environment configuration with TypeScript-safe env variables.

## Epic 02: Authentication

Design a sophisticated JWT-based auth system with 8 granular roles (DEVELOPER, ADMIN, TEACHER, STUDENT, GUARDIAN, ACCOUNTANT, STAFF, USER).
Integrate NextAuth v5 with Prisma adapter, bcryptjs credential auth, and OAuth providers (Google, GitHub) with subdomain-aware callbacks.
Implement role-based access control (RBAC) where JWT tokens carry schoolId and role for automatic route guarding via middleware.
Build password reset flows, two-factor authentication, session management with auto-renewal, and audit logging for sensitive operations.
