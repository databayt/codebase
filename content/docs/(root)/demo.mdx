## Role-Based Views

Each role sees a different dashboard and has different permissions.

| Role | Dashboard | Permissions |
|------|-----------|-------------|
| **Admin** | Full school overview | Create, read, update, delete all entities |
| **Teacher** | Assigned classes | Mark attendance, grade assignments, view students |
| **Student** | Personal schedule | View grades, attendance, submit assignments |
| **Guardian** | Children overview | View linked children's data (read-only) |
| **Staff** | Limited admin | Basic CRUD on assigned entities |
| **Accountant** | Finance focus | Billing and invoices only |

## Demo Limitations

The demo environment resets every 24 hours. Some features are disabled:

- Email notifications (no actual emails sent)
- File uploads (limited to 1MB)
- User creation (cannot create new accounts)
- School settings (locked to demo configuration)
