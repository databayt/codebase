# Gradient Animation

A mesmerizing animated gradient background with interactive cursor tracking. Perfect for hero sections, landing pages, or creating stunning visual effects.

## Installation

No additional dependencies required beyond React and Tailwind CSS.

### Copy the component code

The GradientAnimation component creates animated gradient orbs with customizable colors, blending modes, and interactive cursor tracking.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `gradientBackgroundStart` | `string` | `"rgb(108, 0, 162)"` | Starting color of background gradient |
| `gradientBackgroundEnd` | `string` | `"rgb(0, 17, 82)"` | Ending color of background gradient |
| `firstColor` | `string` | `"18, 113, 255"` | First animated orb color (RGB format) |
| `secondColor` | `string` | `"221, 74, 255"` | Second animated orb color (RGB format) |
| `thirdColor` | `string` | `"100, 220, 255"` | Third animated orb color (RGB format) |
| `fourthColor` | `string` | `"200, 50, 50"` | Fourth animated orb color (RGB format) |
| `fifthColor` | `string` | `"180, 180, 50"` | Fifth animated orb color (RGB format) |
| `pointerColor` | `string` | `"140, 100, 255"` | Interactive cursor color (RGB format) |
| `size` | `string` | `"80%"` | Size of animated orbs |
| `blendingValue` | `string` | `"hard-light"` | CSS blend mode for orbs |
| `interactive` | `boolean` | `true` | Enable cursor tracking |
| `height` | `string` | `"h-screen"` | Height class for container |
| `className` | `string` | - | Additional classes for content wrapper |
| `containerClassName` | `string` | - | Additional classes for main container |
| `children` | `React.ReactNode` | - | Content to display |

## Technical Details

The component creates multiple animated gradient orbs using CSS custom properties and radial gradients. Each orb has its own animation defined in CSS keyframes. The interactive mode uses mouse tracking to create a follow-the-cursor effect with smooth easing.

The component automatically detects Safari browsers and applies appropriate blur filters for optimal rendering across different browsers.

## Color Format

Colors should be provided in RGB format without the `rgb()` wrapper:
- ✅ Correct: `"255, 100, 50"`
- ❌ Incorrect: `"rgb(255, 100, 50)"`

## Performance

The animations use CSS transforms and are GPU-accelerated for smooth performance. The blur effects are optimized differently for Safari and other browsers to ensure consistent visual quality.
